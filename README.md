- [React](https://reactjs.org/)
- [Typescript](https://www.typescriptlang.org/)
- [React Modal](https://github.com/reactjs/react-modal)
- [React Icons](https://react-icons.github.io/react-icons)
- [Axios](https://github.com/axios/axios)
